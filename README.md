# ukr-banks-stability-ml
Classification of Ukrainian commercial banks by financial stability level using machine learning methods



# Bank Clustering Analysis

Цей проєкт присвячений аналізу фінансової стійкості банків за допомогою методу кластеризації `K-Means`.

## Опис проєкту

У роботі використовується набір даних `banks_data.csv`, який містить інформацію про банки та їхні фінансові показники:

- `Bank` — назва банку
- `Total_Assets` — загальні активи
- `Equity` — власний капітал
- `Net_Profit` — чистий прибуток

На основі цих даних обчислюються два аналітичні показники:

- `ROA (Return on Assets)` = `Net_Profit / Total_Assets`
- `Capital_Adequacy` = `Equity / Total_Assets`

Після цього банки групуються на 3 кластери за рівнем фінансової стійкості за допомогою алгоритму `K-Means`.

## Основні можливості

Проєкт виконує такі завдання:

- зчитування та очищення даних;
- видалення дублікатів банків;
- розрахунок показників `ROA` та `Capital_Adequacy`;
- кластеризацію банків методом `K-Means`;
- побудову графіка кластеризації;
- застосування `Elbow Method` для оцінки кількості кластерів;
- розрахунок середніх значень показників по кожному кластеру;
- збереження результатів у CSV-файли;
- додаткову візуалізацію результатів у `R`.

## Структура проєкту

- `main.py` — основний Python-скрипт для аналізу та кластеризації
- `banks_data.csv` — вхідний набір даних
- `banks_clustered.csv` — результат кластеризації банків
- `cluster_means.csv` — середні значення показників по кластерах
- `elbow_method.csv` — дані для методу ліктя
- `cluster_analysis.png` — графік кластеризації та методу ліктя
- `cluster_boxplot.R` — R-скрипт для побудови boxplot
- `cluster_boxplot.png` — додатковий графік розподілу ROA за кластерами
